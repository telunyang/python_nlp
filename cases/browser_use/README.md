## 安裝套件
```bash
# 需要用到 .env 檔案來管理環境變數
pip install python-dotenv

# 安裝 uv
pip install uv

# 安裝 browser-use
uv pip install browser-use

# 安裝用於 browser-use 的瀏覽器
uvx browser-use install
```
